## Animations Observed

- **Hero background**: radial gradient overlay from transparent to `rgb(249, 250, 252)` fades from right edge inward — purely CSS, no JS animation
- **Section fade-in on scroll**: Divi's `et_had_animation` class observed on `.et_pb_row_3` (promotions row) — elements fade up with `opacity: 0 → 1` and `translateY` on scroll entry, approximately 600ms ease-out
- **Section_11 booking CTA**: has `et_animated` class applied — the gold CTA section animates in on scroll with an opacity/translate effect
- **Contact info pill (section_1)**: the navy rounded card appears to slide up or fade in from below on scroll past hero — standard Divi scroll animation
- **Clinic finder**: Agile Store Locator plugin loads map tiles lazily; location list scrolls independently within a fixed-height container
- **FAQ accordion**: click on collapsed item shows `+` icon rotating to `-` and content panel sliding down, ~200ms ease
- **Nav dropdowns (About, Services, Our Clinics)**: dropdown menus appear on hover with `downwards` class — slide down ~150ms
- **Button hover**: gold CTA buttons darken slightly on hover; dark navy buttons lighten; transition ~200ms
- **Card hover (promotion cards)**: subtle box-shadow deepening, ~200ms
